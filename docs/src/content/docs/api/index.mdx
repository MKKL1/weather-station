Heavy Weather's API surface is divided into two distinct components meant for different stages of a device's lifecycle.
We use [Scalar](https://scalar.com) to render our OpenAPI documentation.

You can browse the APIs using the links below, and easily switch between them directly within the documentation UI via
the top-left dropdown!
